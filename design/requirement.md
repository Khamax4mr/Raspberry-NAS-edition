## 기능 요구사항

### 저장소 마운트
- [ ] 사용자가 USB 장치를 라즈베리파이 기기에 연결해서 인식하면, 본 기능을 수행한다.
- [x] USB 장치가 저장소라면, 드라이브 폴더에 새 USB 장치의 저장소 폴더를 만들고 마운트한다.
- [x] 새 USB 장치의 저장소 폴더 이름은 장치의 고유 번호를 바탕으로 만든다.
- [x] 마운트를 성공하면, 드라이브 폴더에 저장소 폴더에 대응하는 [언마운트 파일](#저장소-언마운트)을 만든다.

### 저장소 언마운트
- [x] 언마운트 파일을 실행하면, 본 기능을 수행한다.
- [x] 저장소 폴더가 마운트 상태라면, USB 장치의 마운트를 해제하고 저장소 폴더를 삭제한다.
- [x] 저장소 폴더가 마운트 상태가 아니면, 저장소 폴더를 삭제한다.
- [ ] 관련 없는 다른 USB 장치, 저장소 폴더의 마운트를 해제하지 않아야 한다.


# 비기능 요구사항

## 시스템

### 장비
|유형|사양|
|---|---|
|모델|Raspberry Pi 3B rev 1.2|
|운영체제|Raspberry Pi OS 13|
|아키텍처|aarch64|
|USB Port|4|


### 기술
|기능|사용 기술|
|---|---|
|네트워크 드라이브|Samba|
|JSON 파싱|jq|
|기타|Shell|


## 사용성

### 설치
- [x] 설치 스크립트 파일을 인자와 함께 실행하면 드라이브 요소를 자동으로 구성한다.
- [x] 설치 스크립트 실행에는 `sudo` 권한이 필요하다.
- [x] 드라이브 시스템 동작에 필요한 패키지를 자동으로 설치한다.
- [x] 고정된 경로에 드라이브 폴더를 만든다.

### 연결
- [ ] 드라이브 시스템은 SMB 프로토콜 연결을 지원한다.

### 운영
- [ ] 모든 드라이브 시스템 동작은 드라이브 폴더 안에서만 일어난다.
